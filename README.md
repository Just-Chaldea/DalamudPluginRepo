# Just-Chaldea Dalamud Plugins

A custom Dalamud plugin repository.

## Adding the repository

Open Dalamud settings with `/xlsettings`, go to Experimental, and find Custom Plugin Repositories. Add the URL below, enable it, and save:

```
https://raw.githubusercontent.com/Just-Chaldea/DalamudPluginRepo/main/pluginmaster.json
```

The plugins then appear in the installer (`/xlplugins`) under All Plugins.

## Plugins

### VisitorsPass

Saves housing layouts to JSON files from anywhere you can stand: your own house, a friend's, an apartment, an FC room, or an island sanctuary. Open it with `/visitorspass` or `/vpass`, then Save As to pick a location or Quick Save for one click.

### MahjongAdvisor

Reads the Doman Mahjong board and recommends each move: discard, calls, riichi, and defense.

### CarbunclePartner

Companion to the carbuncleplushy FFXIV Fish Tracker: tracks which fish you have caught, with website-matching filters and one-click export.
